=------------ 事件 ------------=

一个对象可以有多个事件（Events）。事件有以下类：

  controller - 控制器事件
  调用对象内其他的事件或游戏指令，相当于事件内部的触发器。
  
  timer - 计时器事件
  处理事件的所有计时器。
  
  create - 创建事件
  控制器调用的首个对象事件，一般用于初始化的设置。
  
  destroy - 销毁事件
  事件全部结束后，在此进行销毁处理。
  
  user event - 用户事件
  一般的事件，负责除去以上事件的内容。
